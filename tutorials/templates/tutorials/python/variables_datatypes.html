Creating Variables and Using Data Types in Python


Creating Variables in Python:

How to create variables in python:

  • Variable names are lower cased, words separated by underscores, and can include numbers as the first letter.
  • We create variables by setting some word(s) equal (=) to a value
  • Once we set this variable equal to a value in our script, python will do all the work and assign a data type to this newly created variable.

A note on some data types available to us:

Talking through data types:

  • In python we have various different data types which hold different types of values.
  • In this example, we're talking about strings, different number types, and bools, but there are plenty more.
  • Data types are automatically created by the python back end whenever we create the variables in the script. All we have to do is create the value of the variable, python will look at this value, and create a data type is thinks is best fitting to that value
  • We can control this python inferrence however, by using casting. This is just declaring the values data type whenever we create the variable or after variable creation for mutable objects.
