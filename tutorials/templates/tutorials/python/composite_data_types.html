The uses of different composite data types in python:

  • We have access to a couple different composite data sets, and we'll have separate pages for each of these. This is just a page to talk about them all briefly, together.

  • These data types include dictionaries, lists, sets, tuples, and frozensets. The difference among a few of these are mutability. (We'll discuss this more later, but mutable = changeable.)
