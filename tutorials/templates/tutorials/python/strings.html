Introduction to Python Strings


Python Strings, an Array of Values:

Python Strings as Arrays:

  • Python Strings are actually arrays of elements. Each letter is its own value. The entire string then is just the collection of these elements
  • We can see the length of these arrays the same way we can see the length of a composite data type container, by using the len() function.
  • This was designed on purpose this way, as this gives us complete control to very finely be able to edit string type objects.
  • With each letter of a string being it's own value, we can select letters and change them with complete control.

Intentional Arrays, Intentional Control:

Control of Python Strings:

  • Since each string is able to be controlled at such a high level, we can pull out specific values of our strings using indexing
  • In this example we're indexing out the first letter of our string using square brackets and then using the index position, in this example 0, to pull the letters out
  • In the video we only take out one letter, but we can do this with slicing s well, which is method to pull out multiple values.
  • We're also able to do operations on the string as a whole using the str methods that are built into the python package.
