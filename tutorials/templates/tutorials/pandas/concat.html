Pandas Concatenation


Example of concatenating two series together:

Concatenating two series:

  • We have access to the axis keyword which allows us to concatenate along the column or index
  • We can create a longer series from two series, or a dataframe depending on how we concatenate the series
  • Order of the concatenation is important when creating a dataframe (if it matters to you.)

Example of concatenating two DataFrames together:

Concatenating two dataframes:

  • The first dataframe we concatenate with is the dataframe we read in from an external excel file
  • The second dataframe is one in which we create from two pandas series
  • We concatenate these along the axis=1, the column headings
  • We can use the KWARG of 'keys' to create a MultiIndexed data structure object which allows us to pull out the previous dataframes (before the concatenation)
