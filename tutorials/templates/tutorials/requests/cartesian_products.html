Manual Creation of Cartesian Product:

  • We can break down a cartesian product to its rawest form, just the combination of each element in the set in a product set
  • We can get the desired outcome by using multiple for loops
  • This works, however python for loops are notoriously clunky and present a challenge in efficienctly doing this operation whenever the data set size increases.
