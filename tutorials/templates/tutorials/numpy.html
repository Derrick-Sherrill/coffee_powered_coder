NumPy Tutorials

NumPy is one of the first packages any python developer learns. NumPy by itself is useful for linear algebra and matrix computations. NumPy has been integrated into several other packages due to its usefulness in working with data structures.

NumPy Arrays & Data Types
 NumPy Arrays are the building blocks of the entire NumPY package. It's important that we get the concepts down included in this video.

NumPy Indexing
 Indexing gives us the ability to pull out specific data from our NumPy arrays. Through indexing we're able to perform computations only on selected data, add selected data to other data structures, or change values in an array.
