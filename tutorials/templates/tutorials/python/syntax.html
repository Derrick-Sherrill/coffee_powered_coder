Python Tutorial : Syntax and Commenting


Python Commenting:

Talking through comments

  • Comments are denoted with the # symbol.
  • Whenever you place these numbers signs at the beginning of your line, the whole line is not read. If you place them after code on a line, then the code will still be read while the comment will not be.
  • Block quotes should be done how the example shows, a number sign at the beginning of consecutive lines.

Python Syntax:

Talking through syntax

  • Indentations should be done whenever a new block of code depends on the above action.
  • Examples of this are things such as if, while, and for statements.
  • Each block of code should have its own indentation level
  • two nested if statements would mean two nested indentation levels. (8 total spaces over.)
