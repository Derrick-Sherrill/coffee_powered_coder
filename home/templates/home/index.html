Why Learn Python
Python is one of the fastest growing programming languages today. With the ability to make concepts into prototypes very quickly, Python is a great first programming language to learn. There's a large community supporting Python too, ensuring it will be valuable for a long time.

Python Package: NumPy
NumPy is a python packge which is super useful in linear algebra and matrix computations. We can use NumPy arrays to then build up larger data structures when using other packages. Knowing NumPy is a must.

Python Package: TensorFlow (For Machine Learning)
TensorFlow is one of the most well known machine learning python packages. Capable of large scale operations, TensorFlow is a must for AI and predictive algorithms.

Python Package: Pandas
Pandas is a powerful package which allows versatility and ease when dealing with different data structures. The most common of these data structures are pandas dataframes. Dataframes are similar to excel sheets and program most of the same functionality as excel, along with some added benefits of python.
